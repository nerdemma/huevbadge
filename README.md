# 🥚 huevbadge

> Un complemento open-source para **huevsite.io** que extrae dinámicamente tu *Builder Score* y lo convierte en un badge personalizado para tu perfil de GitHub o sitios web.

---

## 🎨 Vista Previa (Preview)

Así es como se verá tu badge oficial utilizando la paleta de colores de la plataforma:

[![Huevsite Score](https://img.shields.io/badge/Score-95-111111?style=for-the-badge&labelColor=ccff00&color=111111&logo=visual-studio-code&logoColor=black)](https://huevsite.io)


---

## 🚀 Características

* ⚡ **Scraping Automatizado:** Utiliza Playwright para saltarse el estado de carga y extraer el score real directamente de la hidratación de Next.js.
* 📦 **API con FastAPI:** Una interfaz web ultra rápida y ligera lista para producción.
* 💾 **Caché Inteligente incorporada:** Guarda las respuestas durante 2 horas en memoria para no sobrecargar los servidores de huevsite y garantizar tiempos de respuesta instantáneos en GitHub.
* 🎨 **Estética Oficial:** Diseñado con los colores exactos de la landing page (Verde Lima Eléctrico `#ccff00` y Negro Mate `#111111`).
